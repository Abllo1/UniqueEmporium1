# UniqueEmporium
An wholesales e-commerce platform where all kinds of shein gowns are sold in bulk for retailers

## Features
- Authentication and user management
- Admin dashboard with analytics
  - Revenue tracking
 - Performance metrics
 - Data visualization (Area, Bar and Pie charts)
 - Date range filtering
- User registration
  - Google athentication
- Payment integration
- Products management
  - Active products listing
  - Products details
  
## Tech Stack
- React.js
- TypeScript
- Supabase as backend and database
- React Query for API data fetching
- Shadcn UI components
- RESTful API integration

## Project Structure
- `public` - Static assets
- `src`- Reactjs directory
  - components-reusabsle and solo UIs
  - context-shared states
  - data
  - hooks
  - pages
  - libs
  - utils
